# 系统架构总览
## 1. 架构定位
本学生学籍管理系统定位为**中小型院校教务管理轻量化系统**，采用B/S架构，支持多终端访问（PC端+移动端适配）。

## 2. 整体架构图
┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
│  客户端层       │      │  应用服务层     │      │  数据存储层     │
│  (Web/移动端)   │─────▶│  (Java Spring)  │─────▶│  (MySQL/Redis)  │
└─────────────────┘      └─────────────────┘      └─────────────────┘

## 3. 核心模块映射
- 学籍管理：依赖学生信息模块 + 权限模块
- 课程管理：依赖课程模块 + 选课关联模块
