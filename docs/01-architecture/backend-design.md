# 后端设计
## 1. 模块划分
backend/
├── user-auth/ # 用户权限模块（登录、角色管理）
├── student-info/ # 学生信息模块（档案、学籍变动）
├── course-manage/ # 课程管理模块（课程维护、选课）
└── common/ # 公共工具模块（数据库连接、加密）

## 2. 技术栈
- 核心框架：Spring Boot 3.2.0
- 持久层：MyBatis-Plus 3.5.4
- 数据库：MySQL 8.0
- 构建工具：Maven 3.6+

## 3. 数据库设计
### 核心表
| 表名         | 功能描述               |
|--------------|------------------------|
| student      | 学生基础信息表         |
| course       | 课程信息表             |
| student_course | 学生选课关联表        |
